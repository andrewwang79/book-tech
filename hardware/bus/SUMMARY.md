# 总线协议

## 串口
| 特性 | RS232 | RS485 |
| - | - | - |
| 电平定义 | 单端信号，高低电平相对于地线。高电平（逻辑1）通常为 -3V 到 -15V，低电平（逻辑0）通常为 +3V 到 +15V。 | 差分信号，电压差决定逻辑状态：A比B电压高时表示逻辑1，A比B电压低时表示逻辑0 |
| 传输距离 | 短（通常15米以内） | 长（可达1200米） |
| 抗干扰能力 | 弱 | 强 |
| 应用特点 | 短距离、点对点通信 | 长距离、多点通信 |
| 应用场景 | 计算机与外设（如打印机、调制解调器）之间的通信 | 工业控制、楼宇自动化、环境监测系统等 |
